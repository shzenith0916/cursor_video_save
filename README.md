# Video Segment Extractor

이 프로그램은 비디오의 특정 구간을 잘라내거나, 비디오에서 이미지를 추출하는 작업을 손쉽게 할 수 있도록 도와주는 GUI 프로그램입니다.

## 주요 기능

- **비디오 자르기**: 타임라인에서 원하는 시작점과 끝점을 선택하여 비디오의 특정 구간을 잘라낼 수 있습니다.
- **이미지 추출**: 비디오의 특정 프레임을 이미지 파일(PNG)로 저장합니다.
- **오디오 추출**: 비디오에서 오디오를 별도 파일(MP3)로 추출합니다.
- **다중 세그먼트 관리**: 여러 개의 비디오 구간을 목록에 추가하고 한 번에 처리할 수 있습니다.

## 요구사항

### 시스템 요구사항
- Python 3.7 이상
- Windows 지원

### 필수 라이브러리
- opencv-python (≥4.8.0)
- numpy (≥1.24.0)
- Pillow (≥10.0.0)
- ttkbootstrap (≥1.10.0)
- python-vlc (≥3.0.20123)

## 설치 안내

### 1. 필수 라이브러리 설치

프로그램 실행에 필요한 파이썬 라이브러리들을 설치합니다.

```bash
pip install -r requirements.txt
```

### 2. FFmpeg 설치

비디오 처리를 위해 FFmpeg가 필요합니다. 아래 방법 중 하나를 선택하여 설치하세요.

- **Windows (winget 사용)**:
  ```bash
  winget install ffmpeg
  ```
- **Windows (Chocolatey 사용)**:
  ```bash
  choco install ffmpeg
  ```
- **macOS (Homebrew 사용)**:
  ```bash
  brew install ffmpeg
  ```
- **Linux (apt 사용)**:
  ```bash
  sudo apt update && sudo apt install ffmpeg
  ```

설치 후, 터미널(명령 프롬프트)에서 `ffmpeg -version` 명령어를 실행하여 설치가 올바르게 되었는지 확인하세요.

FFmpeg가 설치되었는데도 "FFmpeg가 설치되지 않았습니다" 오류가 발생하면 PATH 환경변수를 확인하세요. 또는, 시스템 재시작 후 다시 시도해보세요.

## 사용법

### 🎬 비디오 재생
- 비디오 파일을 선택하여 재생할 수 있습니다
- 재생/정지 버튼으로 비디오 재생을 제어합니다
- 재생 시 오디오가 자동으로 함께 재생됩니다
- 슬라이더를 통해 원하는 위치로 이동할 수 있습니다

### 📹 비디오 추출
- 원하는 구간을 선택하여 비디오 세그먼트를 추출할 수 있습니다
- 시작점과 끝점을 설정하여 정확한 구간 추출이 가능합니다

### 🖼️ 이미지 추출
- 비디오에서 선택한 구간을 이미지로 추출할 수 있습니다
- 프레임 단위로 정밀한 이미지 추출이 가능합니다

### 🎵 오디오 추출
- 비디오에서 오디오를 별도 파일로 추출할 수 있습니다
- MP3 형식으로 고품질 오디오 추출이 가능합니다

### 📋 작업 단계

1. **앱 실행**: `python main.py`로 앱을 실행합니다
2. **비디오 선택**: 파일 메뉴에서 비디오 파일을 선택합니다
3. **재생**: 재생 버튼을 클릭하여 비디오를 재생합니다
4. **구간 저장**: 원하는 구간의 시작점과 끝점을 설정하여 구간을 저장합니다
5. **미리보기**: 설정한 구간을 미리보기로 확인할 수 있습니다
6. **추출**: New 탭으로 이동하여, 저장된 목록에서 구간을 선택한 후 원하는 추출 버튼을 클릭하여 작업을 진행합니다

## 문제 해결

### 일반적인 문제
- **비디오가 재생되지 않을 때**: 지원되는 비디오 포맷인지 확인하세요
- **오디오가 들리지 않을 때**: VLC 미디어 플레이어가 올바르게 설치되었는지 확인하세요
- **추출이 안될 때**: 작업 취소 버튼을 클릭하고 다시 시도하세요
- **FFmpeg 오류**: FFmpeg가 올바르게 설치되었는지 확인하세요

### 지원 포맷
- **비디오**: MP4, AVI, MOV, MKV 등 (OpenCV가 지원하는 포맷)
- **오디오**: MP3, WAV (FFmpeg 기반 추출)
- **이미지**: PNG, JPG (프레임 추출)
